# UniMate 🎓

An AI-powered academic assistant platform designed to optimize student success through predictive analytics and intelligent study management.

UniMate was developed as a comprehensive final-year graduation project for the Software Engineering at The British University in Egypt.

# 🚀 Key Features

AI Performance Prediction: Analyzes academic data to forecast student performance and provide actionable insights.

Custom Quiz Generation: Automatically generates tailored quizzes based on study materials to test knowledge and reinforce learning.

Smart Summaries: Automatically generates concise study summaries to streamline revision.

Deadline Management: Tracks upcoming assignments, exams, and project milestones in a centralized dashboard.

# 🏗️ Architecture & Tech Stack

UniMate is built on a robust layered architecture to ensure scalability and maintainability across all system modules.

Frontend: React.js, TailwindCSS

Backend: Node.js, Express.js

Database:MongoDB

AI: Gemini API

# 🛠️ Installation & Setup

Clone the repository:

Bash
git clone https://github.com/yourusername/unimate.git
cd unimate


Install dependencies:

Bash
Navigate to frontend and install

cd frontend

npm install

Navigate to backend and install

cd backend

npm install

Run the application:

Bash

npm run dev
